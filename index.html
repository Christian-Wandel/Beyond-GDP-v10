What it measures. A country-specific composite (0–100) built from the indicators that matter most for that country's development context, following the UN HLEG "Counting What Counts" framework (2026).

Option B (shown here): Expert-deliberation weights applied to 3–4 priority indicators drawn from the country's own national strategy documents. Base = inequality-adjusted income (Option A); penalties applied for shortfalls relative to global medians.

Option A fallback: When no country-specific config exists, inequality-adjusted household income per capita is shown (HLEG Option A — cross-country comparable).

Reading the score: ≥80 strong, 65–79 good, 50–64 mixed, <50 lagging.
